台股看盤提醒 - Render 雲端版

上線方式：
1. 將 render-cloud 資料夾內容上傳到 GitHub 新專案。
2. 到 https://render.com 建立帳號並登入。
3. New + 選 Web Service。
4. 連接剛剛的 GitHub 專案。
5. 設定：
   - Runtime / Environment: Node
   - Build Command: npm install
   - Start Command: npm start
   - Plan: Free 可先使用
6. 建立後 Render 會給一個網址，例如 https://tw-stock-watch.onrender.com。
7. 手機在外面直接打開該網址即可使用。

提醒：
- 免費版久未使用會休眠，第一次打開可能需要等待約 30 秒。
- 手機設定會存在手機瀏覽器內，和電腦本機版設定分開。
- 本工具資料僅供策略提醒與風險控管參考，不保證投資獲利。
